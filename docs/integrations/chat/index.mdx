# 聊天模型

[聊天模型](/docs/concepts/#chat-models) 是使用一系列 [消息](/docs/concepts/#messages) 作为输入并返回消息作为输出的语言模型（与使用纯文本相对）。这些通常是较新的模型。

:::info

如果您想编写自己的聊天模型，请参见 [这个如何做](/docs/how_to/custom_chat_model/)。
如果您想贡献一个集成，请参见 [贡献集成](/docs/contributing/integrations/)。

:::
