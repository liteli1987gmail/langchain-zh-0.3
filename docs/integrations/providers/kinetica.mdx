# Kinetica

[Kinetica](https://www.kinetica.com/) 是一个实时数据库，专为支持
时间序列和空间数据上的分析和生成式 AI 而构建。

## 聊天模型

Kinetica LLM 包装器使用 [Kinetica SqlAssist
LLM](https://docs.kinetica.com/7.2/sql-gpt/concepts/) 将自然语言转换为
SQL，以简化数据检索的过程。

请参见 [Kinetica 语言到 SQL 聊天模型](/docs/integrations/chat/kinetica) 以获取使用信息。

```python
from langchain_community.chat_models.kinetica import ChatKinetica
```

## 向量存储

Kinetica 向量存储包装器利用 Kinetica 对 [向量相似性搜索](https://docs.kinetica.com/7.2/vector_search/) 的原生支持。
相似性搜索](https://docs.kinetica.com/7.2/vector_search/).

有关用法，请参见 [Kinetica 向量存储 API](/docs/integrations/vectorstores/kinetica)。

```python
from langchain_community.vectorstores import Kinetica
```

## 文档加载器

Kinetica 数据库.
