# Telegram

>[Telegram Messenger](https://web.telegram.org/a/) 是一个全球可访问的免费增值、跨平台、加密、基于云的集中式即时消息服务。该应用程序还提供可选的端到端加密聊天和视频通话、VoIP、文件共享以及其他几个功能。


## 安装和设置

请参阅 [设置说明](/docs/integrations/document_loaders/telegram)。

## 文档加载器

请参阅 [使用示例](/docs/integrations/document_loaders/telegram)。

```python
from langchain_community.document_loaders import TelegramChatFileLoader
from langchain_community.document_loaders import TelegramChatApiLoader
```
