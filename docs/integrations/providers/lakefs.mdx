# lakeFS

>[lakeFS](https://docs.lakefs.io/) 提供可扩展的版本控制，
> 对数据湖进行管理，并使用类似Git的语义来创建和访问这些版本。

## 安装和设置

获取 `ENDPOINT`、`LAKEFS_ACCESS_KEY` 和 `LAKEFS_SECRET_KEY`。
您可以在 [这里](https://docs.lakefs.io/quickstart/launch.html) 找到安装说明。
