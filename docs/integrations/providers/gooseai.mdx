# GooseAI

> [GooseAI](https://goose.ai) 使部署NLP服务变得更简单和更易于访问。
> `GooseAI` 是通过API提供的完全托管的推理服务。
> 与其他知名API具有相同功能的 `GooseAI` 提供即插即用的解决方案
> 以行业最佳经济性服务开源语言模型，只需
> 在您的代码中更改2行。

## 安装和设置

- 使用 `pip install openai` 安装Python SDK
- 从此链接获取您的GooseAI API密钥 [这里](https://goose.ai/)。
- 设置环境变量 (`GOOSEAI_API_KEY`)。

```python
import os
os.environ["GOOSEAI_API_KEY"] = "YOUR_API_KEY"
```
