# CerebriumAI

> [Cerebrium](https://docs.cerebrium.ai/cerebrium/getting-started/introduction) 是一个无服务器的GPU基础设施提供商。
> 它提供对多个大型语言模型的API访问。

请参见[CerebriumAI文档](https://docs.cerebrium.ai/examples/langchain)中的示例。

## 安装和设置

- 安装一个Python包：
```bash
pip install cerebrium
```

- [获取CerebriumAI API密钥](https://docs.cerebrium.ai/cerebrium/getting-started/installation)并设置
它作为环境变量 (`CEREBRIUMAI_API_KEY`)


## 大型语言模型

查看[使用示例](/docs/integrations/llms/cerebriumai)。


```python
from langchain_community.llms import CerebriumAI
```
