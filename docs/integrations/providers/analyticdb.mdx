# AnalyticDB

>[AnalyticDB for PostgreSQL](https://www.alibabacloud.com/help/en/analyticdb-for-postgresql/latest/product-introduction-overview)
> 是一个大规模并行处理 (MPP) 数据仓库服务
> 来自 [阿里云](https://www.alibabacloud.com/)
> 旨在在线分析大量数据。

>`AnalyticDB for PostgreSQL` 基于开源的 `Greenplum Database` 开发
> 项目，并通过 `阿里云` 进行了深入扩展。AnalyticDB
> for PostgreSQL 兼容 ANSI SQL 2003 语法以及 PostgreSQL 和
> Oracle 数据库生态系统。AnalyticDB for PostgreSQL 还支持行存储和
> 列存储。AnalyticDB for PostgreSQL 可以离线处理 PB 级的数据。
> 高性能水平并支持高度并发。

本页面介绍如何在LangChain中使用AnalyticDB生态系统。

## 安装与设置

您需要安装`sqlalchemy` Python包。

```bash
pip install sqlalchemy
```

## 向量存储

查看[使用示例](/docs/integrations/vectorstores/analyticdb)。

```python
from langchain_community.vectorstores import AnalyticDB
```
