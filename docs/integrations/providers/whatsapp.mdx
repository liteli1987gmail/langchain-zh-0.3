# WhatsApp

>[WhatsApp](https://www.whatsapp.com/)（也称为`WhatsApp Messenger`）是一款免费、跨平台的集中式即时消息（IM）和语音通信（VoIP）服务。它允许用户发送文本和语音消息，进行语音和视频通话，并分享图片、文档、用户位置和其他内容。


## 安装和设置

没有任何特殊的设置。
