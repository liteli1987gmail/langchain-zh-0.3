# Apache 软件基金会

> [Apache 软件基金会 (维基百科)](https://en.wikipedia.org/wiki/The_Apache_Software_Foundation)
> 是一个去中心化的开源开发者社区。他们所
> 生产的软件根据 Apache 许可证的条款进行分发，这是一种宽松的
> 开源许可证，适用于自由和开源软件 (FOSS)。Apache 项目
> 的特点是协作、基于共识的开发过程
> 和开放务实的软件许可证，也就是说，它
> 允许自由获得软件的开发者在非自由条款下重新分发
> 它。每个项目由一个自我选择的
> 技术专家团队管理，他们是该项目的活跃贡献者。

## Apache AGE

> [Apache AGE](https://age.apache.org/) 是一个 `PostgreSQL` 扩展，提供
> 图数据库功能。`AGE` 是 `A Graph Extension` 的缩写，
> 受 Bitnine 的 `PostgreSQL 10` 分支 `AgensGraph` 启发，
> 这是一个多模态数据库。该项目的目标是创建一个单一的
> 存储，可以处理关系型和图模型数据，以便用户
> 可以使用标准的 ANSI SQL 以及图查询语言 `openCypher`。
> `Apache AGE` 存储的数据元素是节点、连接它们的边，以及
> 节点和边的属性。
 
了解更多关于 [与 Apache AGE 集成](/docs/integrations/graphs/apache_age) 的信息。

## Apache Cassandra

>[Apache Cassandra](https://cassandra.apache.org/) 是一个 NoSQL、行导向的
> 高度可扩展和高度可用的数据库。从 5.0 版本开始，
> 数据库提供了向量搜索功能。
 
有关 [与 Apache Cassandra 集成](/docs/integrations/providers/cassandra/) 的更多信息，请查看。

## Apache Doris

>[Apache Doris](https://doris.apache.org/) 是一个现代数据仓库，用于
> 实时分析。它在大规模实时数据上提供闪电般快速的分析。
>
> 通常 `Apache Doris` 被归类为 OLAP，并且表现出色
> ClickBench 性能 - 一个分析型数据库管理系统的基准测试。由于它具有
> 超快速的向量化执行引擎，它也可以用作快速的向量数据库。
 
了解更多关于 [与 Apache Doris 集成](/docs/integrations/providers/apache_doris/) 的信息。

了解更多关于 [与 Apache Kafka 集成](/docs/integrations/memory/kafka_chat_message_history) 的信息。


## Apache Spark

> [Apache Spark](https://spark.apache.org/) 是一个统一的分析引擎，
> 用于大规模数据处理。它提供了 Scala、Java 的高级 API，
> Python 和 R，以及一个支持通用计算的优化引擎
> 用于数据分析的图形。它还支持丰富的高级
> 工具，包括用于 SQL 和 DataFrames 的 `Spark SQL`，
> 用于 pandas 工作负载的 `pandas API on Spark`，`MLlib` 用于机器学习，
> `GraphX` 用于图形处理，以及 `Structured Streaming` 用于流式处理。

有关 [与 Apache Spark 集成](/docs/integrations/providers/spark) 的更多信息。
