# Cerebras

在Cerebras，我们开发了世界上最大和最快的AI处理器——晶圆级引擎-3 (WSE-3)。由WSE-3驱动的Cerebras CS-3系统代表了一种新型AI超级计算机，设定了生成式AI训练和推理的标准，具有无与伦比的性能和可扩展性。

作为您的推理提供商，Cerebras可以帮助您：
- 实现AI推理工作负载前所未有的速度
- 以高吞吐量进行商业构建
- 通过我们无缝的集群技术轻松扩展您的AI工作负载

我们的CS-3系统可以快速且轻松地集群，创建世界上最大的AI超级计算机，使得放置和运行最大的模型变得简单。领先的企业、研究机构和政府已经在使用Cerebras解决方案开发专有模型和训练流行的开源模型。

想体验Cerebras的强大功能吗？请查看我们的[网站](https://cerebras.ai)，获取更多资源，并探索通过Cerebras Cloud或本地部署访问我们技术的选项！

有关Cerebras Cloud的更多信息，请访问[cloud.cerebras.ai](https://cloud.cerebras.ai/)。我们的API参考可在[inference-docs.cerebras.ai](https://inference-docs.cerebras.ai/)找到。

## 安装和设置
安装集成包：

```bash
pip install langchain-cerebras
```

## API 密钥
从 [cloud.cerebras.ai](https://cloud.cerebras.ai/) 获取 API 密钥并将其添加到您的环境变量中：
```
export CEREBRAS_API_KEY="your-api-key-here"
```
