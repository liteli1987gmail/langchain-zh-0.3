# Tair

> [阿里云 Tair](https://www.alibabacloud.com/help/en/tair/latest/what-is-tair) 是一个云原生的内存数据库服务
> 由 `阿里云` 开发。它提供丰富的数据模型和企业级能力，以
> 支持您的实时在线场景，同时保持与开源 `Redis` 的完全兼容。
> `Tair` 还引入了基于
> 新型非易失性存储介质 (NVM) 的持久内存优化实例。

## 安装和设置

安装 Tair Python SDK:

```bash
pip install tair
```

## 向量存储

```python
from langchain_community.vectorstores import Tair
```

查看[使用示例](/docs/integrations/vectorstores/tair)。
