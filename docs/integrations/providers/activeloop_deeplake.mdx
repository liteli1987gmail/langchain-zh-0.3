# Activeloop Deep Lake

>[Activeloop Deep Lake](https://docs.activeloop.ai/) 是一个用于深度学习应用的数据湖，允许您将其用作
> 向量存储。

## 为什么选择 Deep Lake？

- 不仅仅是一个（多模态）向量存储。您可以稍后使用该数据集来微调您自己的大型语言模型。
- 不仅存储嵌入，还存储原始数据，并具有自动版本控制。
- 真正的无服务器。不需要其他服务，可以与主要云服务提供商（`AWS S3`、`GCS`等）一起使用。

`Activeloop Deep Lake` 支持 `自查询检索`：
[Activeloop Deep Lake 自查询检索](/docs/integrations/retrievers/self_query/activeloop_deeplake_self_query)
