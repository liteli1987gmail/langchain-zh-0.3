# MLX

> [MLX](https://ml-explore.github.io/mlx/build/html/index.html) 是一个类似于 `NumPy` 的数组框架
> 旨在为 `Apple` 硅芯片上的高效和灵活的机器学习提供支持，
> 由 `Apple 机器学习研究` 提供。


## 安装和设置

安装几个 Python 包：

```bash
pip install mlx-lm transformers huggingface_hub
````


## Chat models


See a [usage example](/docs/integrations/chat/mlx).

```python
from langchain_community.chat_models.mlx import ChatMLX
```
