# TigerGraph

>[TigerGraph](https://www.tigergraph.com/tigergraph-db/) 是一个原生分布式和高性能的图数据库。
> 数据以顶点和边的图形格式存储，导致丰富的关系，
> 非常适合为大型语言模型的响应提供基础。

## 安装和设置

按照说明 [如何连接到 `TigerGraph` 数据库](https://docs.tigergraph.com/pytigergraph/current/getting-started/connection)。

安装 Python SDK：

```bash
pip install pyTigerGraph
```

## 图存储

### TigerGraph

查看[使用示例](/docs/integrations/graphs/tigergraph)。

```python
from langchain_community.graphs import TigerGraph
```
