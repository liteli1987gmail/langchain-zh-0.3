# MindsDB

MindsDB是一个用于从企业数据自定义AI的平台。通过MindsDB及其近200个与[数据源](https://docs.mindsdb.com/integrations/data-overview)和[AI/ML框架](https://docs.mindsdb.com/integrations/ai-overview)的集成，任何开发者都可以利用他们的企业数据更快、更安全地自定义AI以满足他们的需求。

使用MindsDB，您可以将任何数据源连接到任何AI/ML模型，以实现和自动化AI驱动的应用程序。实时部署、服务和微调模型，利用来自数据库、向量存储或应用程序的数据。使用开发者已经熟悉的通用工具完成所有这些。

MindsDB与LangChain集成，使用户能够：


- 在MindsDB中部署通过LangChain提供的模型，使其可访问众多数据源。
- 使用实时和动态数据在MindsDB中微调通过LangChain提供的模型。
- 使用LangChain和MindsDB自动化AI工作流。

请查看[我们的文档](https://docs.mindsdb.com/integrations/ai-engines/langchain)，了解更多关于MindsDB与LangChain集成的信息，并查看示例。
