# Nebula

本页面介绍如何在LangChain中使用[Nebula](https://symbl.ai/nebula)，[Symbl.ai](https://symbl.ai/)的LLM生态系统。
它分为两个部分：安装和设置，以及对特定Nebula包装器的引用。

## 安装和设置

- 获取[Nebula API密钥](https://info.symbl.ai/Nebula_Private_Beta.html)并设置为环境变量`NEBULA_API_KEY`
- 请参阅[Nebula文档](https://docs.symbl.ai/docs/nebula-llm)以获取更多详细信息。

### LLM

存在一个Nebula LLM包装器，您可以通过它访问
```python
from langchain_community.llms import Nebula
llm = Nebula()
```
