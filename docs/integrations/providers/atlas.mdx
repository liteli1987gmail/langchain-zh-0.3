# Atlas

> [Nomic Atlas](https://docs.nomic.ai/index.html) 是一个用于与小型和互联网规模非结构化数据集交互的平台。
> 小型和互联网规模非结构化数据集。


## 安装和设置

- 使用 `pip install nomic` 安装 Python 包
- `Nomic` 也包含在 LangChain 的 poetry 附加功能中 `poetry install -E all`


## 向量存储

查看 [使用示例](/docs/integrations/vectorstores/atlas)。

```python
from langchain_community.vectorstores import AtlasDB
```
