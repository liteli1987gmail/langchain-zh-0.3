# bagelML

> [bagelML](https://www.bagel.net/) (`人工智能开放向量数据库`)，就像是人工智能数据的GitHub。
这是一个协作平台，用户可以创建、
分享和管理向量数据集。它可以支持独立开发者的私人项目，
企业内部协作，以及数据DAO的公共贡献。

## 安装与设置

```bash
pip install bagelML
```


## 向量存储

查看[使用示例](/docs/integrations/vectorstores/bagel)。

```python
from langchain_community.vectorstores import Bagel
```
