# ClickHouse

> [ClickHouse](https://clickhouse.com/) 是一个快速且资源高效的开源数据库，适用于实时应用和分析，支持完整的SQL，并提供广泛的功能来帮助用户编写分析查询。
> 它具有数据结构和距离搜索功能（如 `L2Distance`），以及
> [近似最近邻搜索索引](https://clickhouse.com/docs/en/engines/table-engines/mergetree-family/annindexes)
> 这使得ClickHouse可以作为一个高性能和可扩展的向量数据库，使用SQL存储和搜索向量。
> 这使得 ClickHouse 可以作为一个高性能和可扩展的向量数据库，用于存储和搜索向量，支持 SQL。


## 安装和设置

我们需要安装 `clickhouse-connect` Python包。

```bash
pip install clickhouse-connect
```
