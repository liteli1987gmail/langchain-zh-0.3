# Annoy

> [Annoy](https://github.com/spotify/annoy) (`近似最近邻 Oh Yeah`)
> 是一个 C++ 库，带有 Python 绑定，用于搜索空间中与给定查询点接近的点。
> 它还创建了大型只读文件数据结构，这些数据结构被映射到内存中，以便多个进程可以共享相同的数据。
> 安装和设置

## 安装和设置

```bash
pip install annoy
```


## 向量存储

查看[使用示例](/docs/integrations/vectorstores/annoy)。

```python
from langchain_community.vectorstores import Annoy
```
