# TensorFlow 数据集

> [TensorFlow 数据集](https://www.tensorflow.org/datasets) 是一个可直接使用的数据集集合，
> 可与 TensorFlow 或其他 Python 机器学习框架（如 Jax）一起使用。所有数据集都被暴露为
> [tf.data.Datasets](https://www.tensorflow.org/api_docs/python/tf/data/Dataset)，
> 使得输入管道易于使用且高效。要开始使用，请参见
> [指南](https://www.tensorflow.org/datasets/overview) 和
> [数据集列表](https://www.tensorflow.org/datasets/catalog/overview#all_datasets)。



## 安装和设置

您需要安装 `tensorflow` 和 `tensorflow-datasets` Python 包。

```bash
pip install tensorflow
```

```bash
pip install tensorflow-dataset
```
