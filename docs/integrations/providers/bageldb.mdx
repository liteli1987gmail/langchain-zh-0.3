# BagelDB

> [BagelDB](https://www.bageldb.ai/) (`人工智能开放向量数据库`)，就像是人工智能数据的GitHub。
这是一个协作平台，用户可以创建、
分享和管理向量数据集。它可以支持独立开发者的私人项目，
企业的内部协作，以及数据DAO的公共贡献。

## 安装和设置

```bash
pip install betabageldb
```
