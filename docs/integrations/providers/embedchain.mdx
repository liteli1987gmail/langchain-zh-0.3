# Embedchain

> [Embedchain](https://github.com/embedchain/embedchain) 是一个用于创建 RAG 框架
> 数据管道。它加载、索引、检索和同步所有数据。
>
>它作为一个 [开源包](https://github.com/embedchain/embedchain) 提供
> 以及作为一个 [托管平台解决方案](https://app.embedchain.ai/)。
 

## 安装和设置

使用 pip 安装该包：

```bash
pip install embedchain
```


## 检索器

查看[使用示例](/docs/integrations/retrievers/embedchain)。

```python
from langchain_community.retrievers import EmbedchainRetriever
```
