# E2B

>[E2B](https://e2b.dev/) 提供开源安全沙箱
> 用于AI生成代码的执行。更多信息请见 [这里](https://github.com/e2b-dev)。

## 安装和设置

您需要安装一个Python包：

```bash
pip install e2b_code_interpreter
```

## 工具

查看[使用示例](/docs/integrations/tools/e2b_data_analysis)。

```python
from langchain_community.tools import E2BDataAnalysisTool
```
