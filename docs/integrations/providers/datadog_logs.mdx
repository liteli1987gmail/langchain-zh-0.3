# Datadog 日志

>[Datadog](https://www.datadoghq.com/) 是一个用于云规模应用的监控和分析平台。

## 安装和设置

```bash
pip install datadog_api_client
```

我们必须使用Datadog API密钥和APP密钥初始化加载器，并且需要设置查询以提取所需的日志。

## 文档加载器

请参见[使用示例](/docs/integrations/document_loaders/datadog_logs)。

```python
from langchain_community.document_loaders import DatadogLogsLoader
```
