# 阿里云

>[阿里巴巴集团控股有限公司 (维基百科)](https://en.wikipedia.org/wiki/Alibaba_Group)，或 `阿里巴巴`
> (中文: 阿里巴巴)，是一家专注于电子商务、零售、
> 互联网和技术的中国跨国科技公司。
>
> [阿里云 (维基百科)](https://en.wikipedia.org/wiki/Alibaba_Cloud)，也称为 `阿里云`
> (中文: 阿里云; 拼音: Ālǐyún; 字面意思: '阿里云')，是一家云计算公司，是
> `阿里巴巴集团` 的子公司。`阿里云` 为在线企业提供云计算服务和
> 阿里巴巴自己的电子商务生态系统。
