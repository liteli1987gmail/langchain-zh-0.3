# Postgres 嵌入

> [pg_embedding](https://github.com/neondatabase/pg_embedding) 是一个开源包，用于
> 使用 `Postgres` 和 `层次可导航小世界`
> 算法进行近似最近邻搜索的向量相似性搜索。

## 安装和设置

我们需要安装几个 Python 包。

```bash
pip install psycopg2-binary
```

## 向量存储

查看[使用示例](/docs/integrations/vectorstores/pgembedding)。

```python
from langchain_community.vectorstores import PGEmbedding
```
