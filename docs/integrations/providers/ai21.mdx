# AI21 Labs

>[AI21 Labs](https://www.ai21.com/about) 是一家专注于自然语言处理 (NLP) 的公司，
> 开发能够理解和生成自然语言的 AI 系统。
> 

本页面介绍如何在 `LangChain` 中使用 `AI21` 生态系统。

## 安装和设置

- 获取 AI21 API 密钥并将其设置为环境变量 (`AI21_API_KEY`)
- 安装 Python 包：

```bash
pip install langchain-ai21
```

## 大型语言模型

查看 [使用示例](/docs/integrations/llms/ai21)。

### AI21 大型语言模型

```python
from langchain_ai21 import AI21LLM
```

### AI21 上下文回答

您可以使用 AI21 的上下文回答模型来接收文本或文档，
作为上下文，以及一个问题，并仅基于此上下文返回答案。

```python
from langchain_ai21 import AI21ContextualAnswers
```
