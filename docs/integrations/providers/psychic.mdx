# 灵媒

:::warning
该大模型供应商不再维护，可能无法正常工作。请谨慎使用。
:::

> [灵媒](https://www.psychic.dev/) 是一个与 `Notion`、`Zendesk` 等SaaS工具集成的平台，
> 通过OAuth将这些应用程序中的文档同步到您的SQL或向量
> 数据库。您可以将其视为非结构化数据的Plaid。

## 安装和设置

```bash
pip install psychicapi
```

Psychic 易于设置 - 你只需导入 `react` 库并使用你的 `Sidekick API` 密钥进行配置，该密钥可以从
 [Psychic 控制面板](https://dashboard.psychic.dev/) 获取。当你连接应用程序时，你
可以从控制面板查看这些连接，并使用服务器端库检索数据。
 
1. 在 [控制面板](https://dashboard.psychic.dev/) 创建一个账户。
2. 使用 [react 库](https://docs.psychic.dev/sidekick-link) 将 Psychic 链接模态添加到你的前端 react 应用中。你将使用此功能连接 SaaS 应用。
3. 一旦你创建了连接，你可以通过遵循 [示例笔记本](/docs/integrations/document_loaders/psychic) 使用 `PsychicLoader`。


## 与其他文档加载器的优势

1.	**通用 API:** 不必为每个 SaaS 应用构建 OAuth 流并学习 API，你只需集成一次 Psychic，并利用我们的通用 API 来检索数据。
2.	**数据同步:** 客户的 SaaS 应用中的数据可能会迅速过时。使用 Psychic，你可以配置 Webhook 以每天或实时更新文档。
3.	**简化的 OAuth:** Psychic 处理端到端的 OAuth，这样你就不必花时间为每个集成创建 OAuth 客户端，保持访问令牌的新鲜，并处理 OAuth 重定向逻辑。
