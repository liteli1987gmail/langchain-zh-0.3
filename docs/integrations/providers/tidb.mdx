# TiDB

> [TiDB Cloud](https://www.pingcap.com/tidb-serverless)，是一个综合的数据库即服务 (DBaaS) 解决方案，
> 提供专用和无服务器选项。`TiDB Serverless` 现在正在集成
> 内置的向量搜索到 MySQL 生态中。通过此增强，您可以无缝地
> 使用 `TiDB Serverless` 开发 AI 应用，而无需新的数据库或额外的
> 技术栈。创建一个免费的 TiDB Serverless 集群，并在 https://pingcap.com/ai 开始使用向量搜索功能。
