# 雅虎

> [雅虎 (维基百科)](https://en.wikipedia.org/wiki/Yahoo) 是一家美国网络服务提供商。
>
> 它提供一个网络门户、搜索引擎雅虎搜索，以及相关
> 服务，包括 `我的雅虎`、`雅虎邮箱`、`雅虎新闻`、
> `雅虎财经`、`雅虎体育` 和它的广告平台 `雅虎原生`。
