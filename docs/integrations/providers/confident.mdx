# Confident-ai

>[Confident-ai](https://confident-ai.com) 是 `DeepEval` 的创建者。
>
>[DeepEval](https://github.com/confident-ai/deepeval) 是一个用于单元测试大型语言模型的包。
> 使用 `DeepEval`，每个人都可以通过更快的迭代构建强大的语言模型
> 使用单元测试和集成测试。`DeepEval` 为迭代的每一步提供支持
> 从合成数据创建到测试。

## 安装和设置

您需要获取 [DeepEval API 凭证](https://app.confident-ai.com)。

您需要安装 `DeepEval` Python 包：

```bash
pip install deepeval
```

## 回调

查看[示例](/docs/integrations/callbacks/confident)。

```python
from langchain.callbacks.confident_callback import DeepEvalCallbackHandler
```
