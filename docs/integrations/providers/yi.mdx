# 01.AI

>[01.AI](https://www.lingyiwanwu.com/en)，由李开复博士创立，是一家处于AI 2.0前沿的全球公司。他们提供尖端的大型语言模型，包括从6B到数百亿参数的Yi系列。01.AI还提供多模态模型、开放API平台以及开源选项，如Yi-34B/9B/6B和Yi-VL。

## 安装与设置

在中国网站[这里](https://platform.lingyiwanwu.com/apikeys)或全球网站[这里](https://platform.01.ai/apikeys)注册并获取API密钥。

## 大型语言模型

查看[使用示例](/docs/integrations/llms/yi)。

```python
from langchain_community.llms import YiLLM
```
