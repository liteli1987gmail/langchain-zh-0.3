# Perplexity

> [困惑度](https://www.perplexity.ai/pro) 是搜索互联网的最强大方式
> 通过无限的专业搜索、升级的AI模型、无限文件上传、
> 图像生成和API积分。
>
> 您可以查看[可用模型列表](https://docs.perplexity.ai/docs/model-cards)。

## 安装和设置

安装一个Python包：

```bash
pip install openai
````

Get your API key from [here](https://docs.perplexity.ai/docs/getting-started).
