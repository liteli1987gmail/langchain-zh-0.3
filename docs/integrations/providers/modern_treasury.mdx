# 现代财政

>[现代财政](https://www.moderntreasury.com/) 简化复杂的支付操作。它是一个统一的平台，用于支持移动资金的产品和流程。
>- 连接到银行和支付系统
>- 实时跟踪交易和余额
>- 自动化支付操作以实现规模化
