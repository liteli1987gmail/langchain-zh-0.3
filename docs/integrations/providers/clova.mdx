# Clova

>[CLOVA Studio](https://api.ncloud-docs.com/docs/ai-naver-clovastudio-summary) 是一项服务
> 来自 [Naver Cloud Platform](https://www.ncloud.com/) 的服务，使用 `HyperCLOVA` 语言模型，
> 一种超大规模的人工智能技术，基于用户输入输出通过人工智能技术生成的短语。


## 嵌入模型

请参见 [安装说明和使用示例](/docs/integrations/text_embedding/clova)。

```python
from langchain_community.embeddings import ClovaEmbeddings
```
