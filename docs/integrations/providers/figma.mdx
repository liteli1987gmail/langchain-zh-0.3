# Figma

>[Figma](https://www.figma.com/) 是一个用于界面设计的协作网页应用程序。

## 安装和设置

Figma API 需要一个 `access token`、`node_ids` 和一个 `file key`。

可以从 URL 中提取 `file key`。 https://www.figma.com/file/\/sampleFilename

`Node IDs` 也可以在 URL 中找到。点击任何内容并查找 '?node-id=\' 参数。

`Access token` [说明](https://help.figma.com/hc/en-us/articles/8085703771159-Manage-personal-access-tokens)。
