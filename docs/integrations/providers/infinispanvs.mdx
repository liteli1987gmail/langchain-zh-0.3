# Infinispan VS

> [Infinispan](https://infinispan.org) Infinispan 是一个开源的内存数据网格，提供
> 一个能够存储所有类型数据的键值数据存储，从 Java 对象到纯文本。
> 从版本 15 开始，Infinispan 支持对缓存的向量搜索。

## 安装和设置
请参见 [开始使用](https://infinispan.org/get-started/) 来运行 Infinispan 服务器，您可能想要禁用身份验证
(目前不支持)

## 向量存储

请参见 [使用示例](/docs/integrations/vectorstores/infinispanvs)。

```python
from langchain_community.vectorstores import InfinispanVS
```
