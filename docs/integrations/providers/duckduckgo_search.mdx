# DuckDuckGo 搜索

> [DuckDuckGo 搜索](https://github.com/deedy5/duckduckgo_search) 是一个可以
> 搜索单词、文档、图片、视频、新闻、地图和文本的包
> 使用 `DuckDuckGo.com` 搜索引擎进行翻译。它正在下载文件
> 和图片到本地硬盘。

## 安装和设置

您需要安装一个 Python 包：

```bash
pip install duckduckgo-search
```

## 工具

查看[使用示例](/docs/integrations/tools/ddg)。

有两个可用的工具：

```python
from langchain_community.tools import DuckDuckGoSearchRun
from langchain_community.tools import DuckDuckGoSearchResults
```
