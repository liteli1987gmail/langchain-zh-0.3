# Stack Exchange

>[Stack Exchange](https://en.wikipedia.org/wiki/Stack_Exchange) 是一个涵盖多种领域的问答 (Q&A) 网站网络，每个网站专注于特定主题，
用户在这里可以提出问题、提供答案，并且用户和答案都受到声誉奖励机制的影响。
本页面介绍如何在 LangChain 中使用 `Stack Exchange API`。

## 安装和设置

## 安装与设置
- 安装所需的依赖
```bash
pip install stackapi
```

## 包装器

### 工具

存在一个 StackExchangeAPIWrapper 工具，它包装了这个 API。要导入这个工具：

```python
from langchain_community.utilities import StackExchangeAPIWrapper
```

有关此包装器的更详细说明，请参见 [这个笔记本](/docs/integrations/tools/stackexchange)。

### 工具

您还可以轻松地将此包装器加载为工具（与代理一起使用）。
您可以通过以下方式做到这一点：
```python
from langchain.agents import load_tools
tools = load_tools(["stackexchange"])
```

有关工具的更多信息，请参见 [此页面](/docs/how_to/tools_builtin)。
