# Cloudflare

> [Cloudflare, Inc. (维基百科)](https://en.wikipedia.org/wiki/Cloudflare) 是一家提供
> 内容分发网络服务、云网络安全、DDoS 缓解和 ICANN 认证的
> 域名注册服务的美国公司。

> [Cloudflare Workers AI](https://developers.cloudflare.com/workers-ai/) 允许您通过 REST API 从您的代码在 `Cloudflare` 网络上运行机器
> 学习模型。


## 大型语言模型

请参阅 [安装说明和使用示例](/docs/integrations/llms/cloudflare_workersai)。

```python
from langchain_community.llms.cloudflare_workersai import CloudflareWorkersAI
```

## 嵌入模型

请参阅[安装说明和使用示例](/docs/integrations/text_embedding/cloudflare_workersai)。

```python
from langchain_community.embeddings.cloudflare_workersai import CloudflareWorkersAIEmbeddings
```
