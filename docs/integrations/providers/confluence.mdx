# Confluence

>[Confluence](https://www.atlassian.com/software/confluence) 是一个维基协作平台，用于保存和组织所有与项目相关的材料。`Confluence` 是一个主要处理内容管理活动的知识库。


## 安装和设置

```bash
pip install atlassian-python-api
```

我们需要设置 `用户名/api_key` 或 `Oauth2 登录`。
请参见 [说明](https://support.atlassian.com/atlassian-account/docs/manage-api-tokens-for-your-atlassian-account/)。
