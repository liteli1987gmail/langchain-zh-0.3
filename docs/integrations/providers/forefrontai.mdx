# Forefront AI

> [Forefront AI](https://forefront.ai/) 是一个使您能够
> 微调和推理开源文本生成模型的平台


## 安装和设置

获取一个 `ForefrontAI` API 密钥
访问 [此页面](https://accounts.forefront.ai/sign-in?redirect_url=https%3A%2F%2Fforefront.ai%2Fapp%2Fapi-keys)。
并将其设置为环境变量 (`FOREFRONTAI_API_KEY`)。

## 大型语言模型

查看 [使用示例](/docs/integrations/llms/forefrontai)。

```python
from langchain_community.llms import ForefrontAI
```
