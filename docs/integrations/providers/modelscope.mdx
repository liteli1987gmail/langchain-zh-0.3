# ModelScope

>[ModelScope](https://www.modelscope.cn/home) 是一个大型模型和数据集的仓库。

本页面介绍如何在 LangChain 中使用 modelscope 生态系统。
它分为两个部分：安装和设置，以及对特定 modelscope 包装器的引用。

## 安装和设置

安装 `modelscope` 包。
 
```bash
pip install modelscope
```


## 文本嵌入模型


```python
from langchain_community.embeddings import ModelScopeEmbeddings
```

有关此内容的更详细说明，请参见[此笔记本](/docs/integrations/text_embedding/modelscope_hub)
