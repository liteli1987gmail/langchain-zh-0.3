# DocArray

> [DocArray](https://docarray.jina.ai/) 是一个用于传输中的嵌套、非结构化、多模态数据的库，
> 包括文本、图像、音频、视频、3D网格等。它允许深度学习工程师高效地处理、
> 嵌入、搜索、推荐、存储和传输多模态数据，提供Python风格的API。


## 安装和设置

我们需要安装 `docarray` Python 包。

```bash
pip install docarray
```

## 向量存储

LangChain 提供对 `DocArray` 库中的 `In-memory` 和 `HNSW` 向量存储的访问。

查看 [使用示例](/docs/integrations/vectorstores/docarray_hnsw)。

```python
from langchain_community.vectorstores import DocArrayHnswSearch
```
查看 [使用示例](/docs/integrations/vectorstores/docarray_in_memory)。

```python
from langchain_community.vectorstores DocArrayInMemorySearch
```
