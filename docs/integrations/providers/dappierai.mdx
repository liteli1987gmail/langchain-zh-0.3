# Dappier AI

> [Dappier](https://platform.dappier.com/) 是一个可以访问多样化的
> 实时数据模型的平台。通过 `Dappier` 的预训练、
> 适用于大型语言模型的数据模型增强您的 AI 应用，并确保准确、当前的响应，减少不准确性。

## 安装和设置

要使用其中一个 `Dappier AI` 数据模型，您需要一个 API 密钥。访问
[Dappier 平台](https://platform.dappier.com/) 登录并在您的个人资料中创建一个 API 密钥。
