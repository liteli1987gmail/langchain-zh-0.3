# Label Studio


>[Label Studio](https://labelstud.io/guide/get_started) 是一个开源数据标注平台，为 LangChain 提供了在标注数据以微调大型语言模型 (LLMs) 时的灵活性。它还支持准备自定义训练数据以及通过人类反馈收集和评估响应。

## 安装和设置

请参阅 [Label Studio 安装指南](https://labelstud.io/guide/install) 以获取安装选项。

我们需要安装 `label-studio` 和 `label-studio-sdk-python` Python 包：

```bash
pip install label-studio label-studio-sdk
```


## 回调

查看[使用示例](/docs/integrations/callbacks/labelstudio)。

```python
from langchain.callbacks import LabelStudioCallbackHandler
```
