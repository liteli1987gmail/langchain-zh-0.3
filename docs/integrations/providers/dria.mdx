# Dria

>[Dria](https://dria.co/) 是一个供开发者使用的公共 RAG 模型中心，
> 既可以贡献也可以利用共享的嵌入湖。

查看有关 LangChain 与 Dria 集成的更多细节
在 [此页面](https://dria.co/docs/integrations/langchain)。

## 安装和设置

您需要安装一个 Python 包：

```bash
pip install dria
```

您必须从Dria获取API密钥。您可以通过在[Dria](https://dria.co/)注册来获取它。

## 检索器

请参见[使用示例](/docs/integrations/retrievers/dria_index)。

```python
from langchain_community.retrievers import DriaRetriever
```
