# PromptLayer

>[PromptLayer](https://docs.promptlayer.com/introduction) 是一个用于提示工程的平台。
> 它还帮助 LLM 可观察性，以可视化请求、版本提示并跟踪使用情况。
>
>虽然 `PromptLayer` 确实有与 LangChain 直接集成的 LLM（例如。
> [`PromptLayerOpenAI`](https://docs.promptlayer.com/languages/langchain)），
> 使用回调是将 `PromptLayer` 与 LangChain 集成的推荐方式。

## 安装和设置

要使用 `PromptLayer`，我们需要：
- 创建一个 `PromptLayer` 账户
- 创建一个API令牌并将其设置为环境变量（`PROMPTLAYER_API_KEY`）

安装一个Python包：

```bash
pip install promptlayer
```


## 回调

查看 [使用示例](/docs/integrations/callbacks/promptlayer)。

```python
import promptlayer  # Don't forget this import!
from langchain.callbacks import PromptLayerCallbackHandler
```

查看 [使用示例](/docs/integrations/llms/promptlayer_openai)。
