# Arcee

> [Arcee](https://www.arcee.ai/about/about-us) 使开发和进步成为可能
> 我们称之为 SLMs——小型、专业、安全和可扩展的语言模型。
> 通过提供 SLM 适配系统和无缝、安全的集成，
> `Arcee` 使企业能够充分利用
> 领域适应的语言模型，推动变革性
> 运营创新。


## 安装和设置

获取您的 `Arcee API` 密钥。

查看[使用示例](/docs/integrations/llms/arcee)。

```python
from langchain_community.llms import Arcee
```
