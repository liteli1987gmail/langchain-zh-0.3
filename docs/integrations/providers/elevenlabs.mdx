# ElevenLabs

> [ElevenLabs](https://elevenlabs.io/about) 是一家语音AI研究与部署公司
> 其使命是使内容在任何语言和声音中都能普遍可及。
>
> `ElevenLabs` 创建了最真实、多功能和具有上下文意识的
> AI音频，提供在29种语言中生成数百种
> 新的和现有的声音的能力。

## 安装和设置

首先，您需要设置一个ElevenLabs账户。您可以按照
[此处的说明](https://docs.elevenlabs.io/welcome/introduction)进行操作。

安装Python包：

```bash
pip install elevenlabs
```

## 工具

查看[使用示例](/docs/integrations/tools/eleven_labs_tts)。

```python
from langchain_community.tools import ElevenLabsText2SpeechTool
```
