# 维基百科

>[维基百科](https://wikipedia.org/) 是一个由志愿者社区（称为维基人）编写和维护的多语言免费在线百科全书，通过开放协作和使用一种名为MediaWiki的基于维基的编辑系统。`维基百科` 是历史上最大和阅读量最高的参考书籍。


## 安装和设置

```bash
pip install wikipedia
```
