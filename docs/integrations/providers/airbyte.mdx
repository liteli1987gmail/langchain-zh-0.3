# Airbyte

>[Airbyte](https://github.com/airbytehq/airbyte) 是一个用于从API、
> 数据库和文件到数据仓库和数据湖的ELT管道的数据集成平台。它拥有最大的ELT连接器目录，支持数据仓库和数据库。

## 安装和设置

```bash
pip install -U langchain-airbyte
```

:::note

目前，`langchain-airbyte`库不支持Pydantic v2。
请降级到Pydantic v1以使用此包。

此包目前还需要Python 3.10及以上版本。

:::

集成包不需要任何需要设置的全局环境变量，但某些集成（例如`source-github`）可能需要传入凭据。
