# 问候设备

问候设备提供 GigaChat 大型语言模型的模型。

有关如何获取 GigaChat 访问权限的更多信息，请 [点击这里](https://developers.sber.ru/docs/ru/gigachat/api/integration)。

## 安装和设置

可以通过 pip 从 PyPI 安装 GigaChat 包：

```bash
pip install gigachat
```

## 大型语言模型

查看[使用示例](/docs/integrations/llms/gigachat)。

```python
from langchain_community.llms import GigaChat
```

## 聊天模型

查看[使用示例](/docs/integrations/chat/gigachat)。

```python
from langchain_community.chat_models import GigaChat
```

## 嵌入

查看[使用示例](/docs/integrations/text_embedding/gigachat)。

```python
from langchain_community.embeddings import GigaChatEmbeddings
```
