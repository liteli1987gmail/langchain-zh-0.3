# 检索器

一个 [检索器](/docs/concepts/#retrievers) 是一个接口，它根据非结构化查询返回文档。
它比向量存储更为通用。
检索器不需要能够存储文档，只需返回（或检索）它们。
检索器可以从向量存储创建，但也足够广泛，包括 [维基百科搜索](/docs/integrations/retrievers/wikipedia/) 和 [亚马逊 Kendra](/docs/integrations/retrievers/amazon_kendra_retriever/)。

检索器接受字符串查询作为输入，并返回一个 [文档](https://python.langchain.com/api_reference/core/documents/langchain_core.documents.base.Document.html) 列表作为输出。

有关如何使用检索器的具体信息，请参见 [相关的使用指南](/docs/how_to/#retrievers)。

请注意，所有 [向量存储](/docs/concepts/#vector-stores) 都可以 [转换为检索器](/docs/how_to/vectorstore_retriever/)。
有关可用向量存储的信息，请参阅向量存储 [集成文档](/docs/integrations/vectorstores/)。
此页面列出了通过子类化 [BaseRetriever](/docs/how_to/custom_retriever/) 实现的自定义检索器。
