# 大型语言模型

:::caution
您当前正在查看关于[文本补全模型](/docs/concepts/#llms)的文档。许多最新和最受欢迎的模型是[聊天补全模型](/docs/concepts/#chat-models)。

除非您正在使用更高级的提示技术，否则您可能想查看[此页面](/docs/integrations/chat/)。
:::

[大型语言模型](/docs/concepts/#llms)是将字符串作为输入并返回字符串作为输出的语言模型。

:::info

如果您想编写自己的大型语言模型，请参见[此如何做](/docs/how_to/custom_llm/)。
如果您想贡献一个集成，请参见[贡献集成](/docs/contributing/integrations/)。

:::
