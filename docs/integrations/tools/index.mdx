# 工具

[工具](/docs/concepts/#tools) 是设计用于被模型调用的实用程序：它们的输入旨在由模型生成，输出旨在返回给模型。

[工具包](/docs/concepts#toolkits) 是一组旨在一起使用的工具的集合。

:::info

如果您想编写自己的工具，请参见[这个如何做](/docs/how_to/custom_tools/)。
如果您想贡献一个集成，请参见[贡献集成](/docs/contributing/integrations/)。

:::
