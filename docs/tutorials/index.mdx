## LangGraph

LangGraph 是 LangChain 的一个扩展，旨在通过将步骤建模为图中的边和节点，构建强大且有状态的多参与者应用程序，使用大型语言模型 (LLMs)。
LangGraph 文档目前托管在一个单独的网站上。

您可以在这里查看 [LangGraph 教程](https://langchain-ai.github.io/langgraph/tutorials/)。
您可以在[这里查看LangGraph教程](https://langchain-ai.github.io/langgraph/tutorials/)。

## LangSmith

LangSmith 允许您密切跟踪、监控和评估您的 LLM 应用程序。
它与 LangChain 无缝集成，您可以在构建时检查和调试链的各个步骤。

LangSmith 文档托管在一个单独的网站上。
您可以在这里查看 [LangSmith 教程](https://docs.smith.langchain.com/tutorials/)。
